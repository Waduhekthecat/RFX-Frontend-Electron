## RFX – UI/UX Design Exploration

This folder contains early UI/UX design explorations created in Figma for the RFX audio routing platform.

## Project Context

RFX is a real-time audio routing system designed to function like a digital pedalboard (similar to Line 6 Helix / Neural DSP Quad Cortex), using a DAW as the audio engine.

## Design Goals
Create a clear and intuitive signal flow
Mirror familiar guitar pedal workflows
Minimize cognitive load during real-time interaction
Enable fast control of routing, FX chains, and parameters

## Key Screens

Routing Screen (routing-screen.png)

Designed to visually represent signal flow in a way that matches real-world pedal chains
Focused on clarity and quick understanding of routing paths

EQ / Parameter Control (eq.png)

Structured controls to group related parameters logically
Designed for fast adjustments during live use

Main Edit Screens (main-edit-screens.png)

Explored layout hierarchy and grouping of controls
Prioritized readability and quick navigation

Design Drafts (design-drafts.png)

Early layout explorations used to test different UI structures and interaction patterns
Design Approach
Focused on usability over visual complexity
Iterated on layouts to reduce friction in real-time workflows
Designed interfaces to feel familiar to musicians rather than traditional software users
